# Medical-Image-Analysis-using-Deep-Neural-Network-from-Radiology-Images
A deep learning–based medical image analysis project for detecting lung diseases, especially Tuberculosis (TB), from chest X-ray images. It uses CNN models and a Teacher–Student framework to achieve high accuracy with efficient performance, supporting fast and reliable diagnostic predictions.

## Features

- Deep learning–based detection of lung diseases from chest X-ray images
- Focused Tuberculosis (TB) classification using CNN models
- Teacher–Student framework for improved accuracy with reduced computational cost
- Image preprocessing and augmentation for better model generalization
- Fast and efficient prediction for real-time diagnostic support

  ## Technologies Used

 - Python
-  TensorFlow / Keras
-  OpenCV
-  NumPy, Pandas
-  Matplotlib
-  Google Colab / Jupyter Notebook
